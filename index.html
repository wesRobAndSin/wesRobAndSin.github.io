About me

I am a software developer with experience in Ruby, Ruby on Rails, and PostgreSQL on the back end, and ReactJS with a Redux architectural framework on the front end. Formerly a trader, I love to solve problems and uncover patterns, not only in my work but also in life.

NotOKCupid

NotOKCupid is a full-stack web application inspired by OKCupid, with a twist. NotOKCupid's matching algorithm matches users based on their incompatibility, because sometimes the worst love matches turn out to be the best. It utilizes Ruby on Rails on the backend, a PostgreSQL database, and React.js with a Redux architectural framework on the frontend.

Stick Tennis

A clone of the classic Atari Pong game. It uses jQuery to manipulate elements on the page, while using the HTML Canvas element to render the actual game play. Use the up and down arrow keys to to try to beat the computer AI!

Query Buster

A lightweight ORM library that utilizes metaprogramming for back end database management. Comes equipped with a demo that is playable in the terminal. Feel free to clone on github and follow the instructions to try it out!
